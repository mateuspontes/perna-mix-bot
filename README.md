# Perna Mix Bot

A Discord bot for creating balanced teams for games.

## Running Locally

1. Install [Rust](https://rustup.rs/)

2. Set environment variable:
```bash
export DISCORD_TOKEN="your_discord_bot_token"
```

3. Run the bot:
```bash
cargo run
```

The bot will start on port 10000. Health check available at `http://localhost:10000/healthcheck`.
